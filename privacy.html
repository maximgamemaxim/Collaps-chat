Политика конфиденциальности Collaps Chat

Последнее обновление: 19 марта 2026 г.

1. Какие данные мы собираем

При регистрации и использовании Collaps Chat мы собираем следующую информацию:

  • Имя пользователя (никнейм) — для отображения в чате
  • Юзернейм для поиска — для поиска вас другими пользователями
  • Email — для входа в аккаунт и восстановления пароля
  • Пароль — хранится в зашифрованном виде
  • Сообщения — которые вы отправляете в чатах, каналах и группах
  • Время последнего визита и статус онлайн — для отображения активности

2. Как мы используем ваши данные

  • Для обеспечения работы мессенджера (вход, отправка сообщений)
  • Для отображения вашего профиля другим пользователям
  • Для модерации контента (проверка на запрещённые слова)
  • Для технической поддержки

3. Где хранятся данные

Все данные хранятся в Firebase (Google Cloud), серверы находятся в Европе (регион europe-west1). Firebase соответствует международным стандартам безопасности (ISO 27001, SOC 1/2/3).

4. Передача данных третьим лицам

Мы НЕ передаём ваши персональные данные третьим лицам. Исключение: только по запросу государственных органов в соответствии с законодательством РФ.

5. Защита данных

  • Пароли хранятся в зашифрованном виде
  • Доступ к базе данных защищён правилами Firebase Security
  • Мы используем HTTPS для шифрования трафика

6. Ваши права

Вы имеете право:

  • Запросить копию всех ваших данных (напишите на collapschat@bk.ru)
  • Исправить неточные данные (в настройках профиля)
  • Удалить свой аккаунт и все данные (кнопка в настройках или письмо на support)
  • Отозвать согласие на обработку данных (удалив аккаунт)

7. Удаление аккаунта

Чтобы удалить аккаунт и все связанные с ним данные:

  • Способ 1: Войдите в настройки и нажмите кнопку "Удалить аккаунт" в разделе "Опасная зона"
  • Способ 2: Напишите письмо на collapschat.legal@bk.ru с вашего email, указанного при регистрации
  • Способ 3: Напишите в чат поддержки (@support)

8. Изменения в политике

Мы можем обновлять эту политику. При значительных изменениях мы уведомим вас через сайт или по email.

9. Контакты

По всем вопросам, связанным с персональными данными, пишите:

  • Email: collapschat.legal@bk.ru
  • Чат с поддержкой: @support в Collaps Chat
